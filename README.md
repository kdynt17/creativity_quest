# creativity_quest

이진 탐색을 배우기 위한 동기부여 문제입니다. 모양이 같은 8개의 추 중 1개만 가볍고, 저울을 최대 3번 사용해 정답을 찾는 웹 게임입니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

## 조작

- 추를 드래그해서 왼쪽 또는 오른쪽 저울 접시에 올립니다.
- 양쪽에 같은 개수의 추를 올린 뒤 `측정`을 누릅니다.
- 결과를 바탕으로 정답 칸에 추 1개를 놓고 `정답`을 누릅니다.
- 정답이면 축하 폭죽이 나오고, 오답이거나 3번을 모두 쓰면 재시작해서 다시 도전합니다.
